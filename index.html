Decentralized storage of annotations

Intermediate thesis presentation by Lukas Vanhoucke

Current social media networks

  • All data is stored in a centralized manner
    • EULA usually states that you do not own the stored data
    • Need to create new account and data for each social network (information silos)
    • Granularity of access rights to data is dependent on social network
  • Annotation systems are a kind of social network, since it involves users interacting with eachother.
  • For practical reasons, annotation systems are almost always centralized.
  • Is it possible to decentralize annotation systems to solve the issues stated above?
